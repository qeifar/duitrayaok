# duitrayaok

Track and total **Duit Raya** money packets **per child** — static **PWA** for **GitHub Pages**. Data in `localStorage` (`duitrayaok_v1`).

## Local preview

```bash
python3 -m http.server 8080
```

Use `http://127.0.0.1:8080` so the service worker can register.

## GitHub

Enable **Pages** on this repo: **Settings → Pages →** branch **`main`**, folder **`/` (root)**.

Change the **Donate** link in `index.html` if needed.
